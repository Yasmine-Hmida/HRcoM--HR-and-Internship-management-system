# HRcoM - HR & Internship management system

## Overview

𝗛𝗥𝗰𝗼𝗠 is a web platform that covers the main operations of the HR department (user management, leave management, alert management, and more), while also integrating a way to mark your presence with the facial identification and a payroll management module compliant with the current finance law.

## Technologies Used

- ReactJS (For the Frontend).
- NodeJS, ExpressJS (For the Backend).
- MongoDB (For the Database).
- Socket.IO (For emitting real-time events in the platform).
